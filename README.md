# Power-BI-Project-on-Sudamerican-Represetation-on-Movies-Shows-in-Netflix
-- Power BI Project on Sudamerican Represetation on Movies/ Shows in Netflix
-- Power BI Project done as a research project for a Power BI and Excel course.
-- The instruction was to do a visual representation of whatever topic we wanted to research.
-- I chose a Dataset which contained all the Netflix catalogue until 2021.
-- I transformed the data on PowerQuery and then did a general visualization where you can sort the respective titles by country and realeasing date.
-- The I added a pair of visualizers signaling titles from SudAmerican countries and their respective genres onto Netflix.
